# DeepFrag2Mol
Converting the molecular fragments generated by DeepFrag into complete molecules

## 编译后的二进制文件




## 安装

方法1： 源码安装

git clone

cd 
conda env create -f deepfrag2mol.yaml

conda activate deepfrag2mol
